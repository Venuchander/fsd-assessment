# FSD Assessment

- Part 1: React + TypeScript
  - `useLocalStorage`
  - `useDebounce`
  - Reusable Button component

- Part 2: Node.js + Express + TypeScript
  - RBAC middleware
  - Protected routes

## Run

### Part 1

```bash
cd "Part 1"
npm i
npm run dev
```

Runs on `http://localhost:5173`

### Part 2

```bash
cd "Part 2"
npm i
npm run dev
```

Runs on `http://localhost:3000`
